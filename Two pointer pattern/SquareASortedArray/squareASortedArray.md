## Square a Sorted Array

You're given a sorted array of positive and negative numbers. Write a function that returns a new array containing squares of all the numbers of the input array in a sorted order. 

<pre>
<h3>Sample Input</h3>
[-4, -2, 0, 3, 5]
<h3>Sample Output</h3>
[ 0, 4, 9, 16, 25 ]
</pre>

<pre>
<h3>Sample Input</h3>
[-3, -2, 0, 1, 2]
<h3>Sample Output</h3>
[ 0, 1, 4, 4, 9 ]
</pre>
